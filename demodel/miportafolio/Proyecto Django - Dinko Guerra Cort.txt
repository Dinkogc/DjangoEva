Proyecto Django - Dinko Guerra Cortés

instrucciones. Anotar en la terminal de VSC

1. Acceder al directorio del proyecto ==> cd miportafolio
2. Instalar Babel (para formatear la fecha y hora de manera localizada en español) ==> pip install babel
3. Aplicar las migraciones ==> python manage.py migrate
4. Iniciar el servidor ==> Python manage.py runserver
